本项目想基于LBS，并结合社交网络的理念，为你找到身边最需要帮助的人，并标记出那些利用利用大家爱心牟利的人，尽可能的帮助需要的人。
基于LBS和社交网络的随手慈善网站，客户端基于微信和支付宝公共主页，使得周边的人都可以随手帮助到身边的人，也可以利用设计网络的特性，使得帮助得以放大。
